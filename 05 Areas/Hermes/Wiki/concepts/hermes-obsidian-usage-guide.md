# Obsidian Usage Guide

## คืออะไร
แนวทางการจัดหมวดหมู่โฟลเดอร์ใน Obsidian vault สำหรับ Hermes ecosystem

## โครงสร้าง vault
| โฟลเดอร์ | หน้าที่ |
|-----------|---------|
| `01 Inbox` | กล่องพักโน้ตชั่วคราว ยังไม่จัดหมวด |
| `02 Core` | กฎหลัก ตัวตน วิธีทำงานที่ต้องยึดตลอด |
| `03 Daily` | บันทึกรายวัน |
| `04 Projects` | งานที่มีจุดจบชัด |
| `05 Areas` | เรื่องที่ต้องดูแลต่อเนื่อง |
| `06 Resources` | ความรู้และข้อมูลอ้างอิง |
| `07 MOCs` | หน้าสารบัญหรือหน้ารวมลิงก์ |
| `08 Archive` | ของเก่า เก็บไว้เฉย ๆ |
| `09 Templates` | แม่แบบสำหรับสร้างโน้ตใหม่ |

## กฎง่าย ๆ
- **ความรู้อ้างอิง** → `06 Resources`
- **เรื่องดูแลต่อเนื่อง** → `05 Areas`
- **งานมีเป้าหมายและจบได้** → `04 Projects`
- **โน้ตชั่วคราว** → `01 Inbox` ก่อน
- **หน้ารวมทางลัด** → ทำเป็น MOC
