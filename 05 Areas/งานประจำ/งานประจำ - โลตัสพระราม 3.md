# งานประจำ - โลตัสพระราม 3

## ข้อมูลหลัก
- สถานที่ทำงาน: โลตัสพระราม 3
- ตำแหน่ง/หน้าที่: พนักงาน PC จัดเรียงสินค้า Thaibeverage
- เวลาเข้างาน: 09:00–18:00
- วันหยุดประจำสัปดาห์: วันจันทร์
- 601 ชื่อ พี่หนุ่ย
- 503 ชื่อ พี่เติ้ล

## รายละเอียดงาน
> พี่จอนจะทยอยให้เฮอช่วยจำรายละเอียดงานประจำเพิ่มเติมตรงนี้

### สรุปประชุมเรื่องพื้นที่เช่า / OOS / การแจ้งความเคลื่อนไหว

#### 1) พื้นที่เช่า
- พื้นที่เช่าเกิด/ไม่เกิด ต้องทำในระบบก่อน 12:00 น.
- พื้นที่เช่าในระบบ ต้องทำให้เสร็จภายในวันนั้น ไม่เกิน 17:00 น.
- พื้นที่เช่าต้องตรงจุดเช่า และสินค้าต้องตรงตามแบรนด์สินค้า
- ถ้ามี Layout แต่สินค้าไม่พอ ต้องถ่ายสต๊อกส่งมาด้วย
- ถ้ามี Layout แต่สาขาเอาพื้นที่ไปใช้ทำกิจกรรมอื่น ต้องระบุเหตุผลให้ครบ
- ถ้าไม่มี Layout ในสาขา ต้องระบุชื่อ Layout และแจ้งว่าไม่มีในสาขา
- ถ้าสินค้าตั้งกองแต่ไม่พอทำฐาน:
  - ถ้ามีล้อมกอง ให้ระบุว่าฐานไม่ใช่สินค้า
  - ถ้าไม่มีล้อมกอง ให้ระบุว่าฐานเป็นสินค้าอะไร

#### 2) การทำ OOS
- รายงาน OOS ประจำวัน ให้ทำช่วงก่อนออกจากสาขา
- FM ประจำสาขา ให้ทำช่วง 17:00–19:00 น. ก่อนเลิกงาน
- พนักงานที่วิ่งหลายสาขา ให้ทำก่อนออกจากสาขาทุกครั้ง
- เวลาลง OOS ต้องใส่หมายเหตุให้ละเอียดและเป็นมาตรฐาน เช่น
  - OOS เดี่ยว / OOS ลัง
  - สินค้าไม่เข้าตามรอบ
  - สินค้าค้าง OH ในระบบ
  - สาขาเพิ่งรับ OH / ปรับเรียบร้อยแต่ยัง OOS
  - สินค้าติด Provision
  - สินค้าหลุดแพลนในโปรแกรม
- หมายเหตุควรระบุให้ครบ เช่น กำหนดสินค้าเข้าวันที่อะไร จำนวนเท่าไร มีการ Order Review วันไหน และติดต่อใคร

#### 3) การแจ้งความเคลื่อนไหว
- ต้องแจ้งสินค้า NPD ของบริษัทและคู่แข่ง
- ต้องแจ้งสินค้าเปลี่ยนแพ็กเกจใหม่ของบริษัทและคู่แข่ง
- ต้องแจ้งสินค้าปรับราคา
- ต้องแจ้งสื่อและของแถมต่าง ๆ ที่เข้ามาในสาขา
- ต้องแจ้งกิจกรรมต่าง ๆ ในสาขา

#### 4) หลักเกณฑ์พื้นที่พิเศษ / ค่าพื้นที่พิเศษ
- ใช้เกณฑ์ขนาดพื้นที่ (Size-based) และระยะเวลา (Period) เป็นเงื่อนไขจ่ายเงิน
- Size S: 3–10 ลัง
  - 7 วัน = 150 บาท
  - 14 วัน = 300 บาท
- Size M: 11–30 ลัง
  - 7 วัน = 250 บาท
  - 14 วัน = 500 บาท
- Size L: 30 ลังขึ้นไป
  - 7 วัน = 350 บาท
  - 14 วัน = 700 บาท
- จะจ่ายตามพื้นที่ที่เกิดจริงและจัดเรียงตามที่กำหนด
- ตัวอย่างตำแหน่งพื้นที่ที่ใช้ได้: หน้าทางเดินเข้า, หน้าแผนก, ตู้แช่, หน้า–หลังแคชเชียร์
- เป้าหมายของพื้นที่พิเศษคือ เพิ่มพื้นที่การจัดเรียงสินค้า, สร้างการรับรู้, ป้องกันพื้นที่ขายจากสินค้าคู่แข่ง, และเพิ่มโอกาสในการสร้างยอดขาย

#### สิ่งที่ต้องจำแบบเร็ว
- พื้นที่เช่าเกิด/ไม่เกิด: ก่อน 12:00 น.
- พื้นที่เช่าในระบบ: ไม่เกิน 17:00 น.
- OOS: ทำก่อนออกจากสาขา
- ถ้าต้องลงเหตุผลหรือหมายเหตุ ให้ลงแบบละเอียด ไม่พิมพ์สั้นกำกวม
