# Obsidian + Hermes Quickstart

## ใช้ทำอะไร
- Obsidian = long-term memory / knowledge base
- Hermes = ผู้ช่วยอ่าน เขียน ค้น สรุป และจัดระเบียบ
- MCP = สะพานเชื่อม Hermes กับ vault

## คำสั่งที่ใช้บ่อยกับเฮอ
- “จดอันนี้ลง Obsidian ใน Inbox”
- “สรุปแชทนี้ลง Daily Note วันนี้”
- “ค้นใน Obsidian ว่าเคยจดเรื่อง X ไหม”
- “สร้าง MOC สำหรับหัวข้อ X”
- “ทำ weekly review จาก daily notes สัปดาห์นี้”

## Best Practices
- เก็บ secrets ออกจาก vault
- ให้ backup ก่อนเขียนเยอะ ๆ
- ใช้ Daily Notes เป็น inbox รายวัน
- ใช้ MOC เป็นหน้าสารบัญ
- ให้ Hermes สรุป diff/แผนก่อนแก้โน้ตสำคัญ
